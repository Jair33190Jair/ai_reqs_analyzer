GF-1F604CCRITICALFINDINGAMBIGUITYconfidence 95%

Item: SYS-FUNC-003

SYS-FUNC-003 uses 'real-time' without a numeric bound. The term is undefined in the document and two engineers would assign different timing budgets. Additionally, 'avoid' does not specify minimum clearance distance, detection range, or the maximum speed at which avoidance must be effective.

Recommendation: Replace 'real-time' with a concrete response time (e.g., 'within 200 ms of detection'). Add minimum clearance distance (e.g., '≥ 10 cm from detected obstacles'). Specify the speed envelope over which avoidance must operate.

Reference: ISO/IEC/IEEE 29148:2018 §6.2.5.2; ASPICE SYS.1.BP2

Item: SYS-FUNC-001
